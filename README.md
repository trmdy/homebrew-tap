# Homebrew Tap

Homebrew tap for trmdy's projects.

## Installation

```bash
brew tap trmdy/tap
```

## Available Casks

| Cask | Description |
|------|-------------|
| `mac-bar-timer` | Menu bar timer for macOS focused on long work sessions |

### Mac Bar Timer

```bash
brew install --cask trmdy/tap/mac-bar-timer
```
